# Project Examples
### Project: tfmod-proxmox-kvm-debian13-cli
There are 3 examples included in this folder for the project:
1. ansible-webnode
    - Uses Ansible to install a comprehensive LAMP stack and PHP Wordpress Site.
2. bash-ahc
    - Uses BASH scripts to install a simplerPython CLI application.
3. docker-dockhand
    - Uses Docker Compose tools to install Postgresql and Dockhand, allowing you to host and manage more OCi/Docker Containers on Proxmox using this VM. Size this VM accordingly.
